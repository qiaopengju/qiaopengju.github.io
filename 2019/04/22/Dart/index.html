操作符 解释
~/ 除号，返回整数
== 定义在左边对象上的函数，用于判等
as 类型转换
is 如果对象是指定的类型返回 True
is! 如果对象是指定的类型返回 False
??= 若变量为null，则赋值，否则不变
.. 级联操作符，用以访问一个方法返回值的成员变量
condition ?? expr1 : expr2; 特殊算数表达式，若condition为null则执行expr1
?. 和 . 类似，但是左边的操作对象不能为 null。例如 foo?.ba 如果foo 为 null 则返回 null，否则返回bar
